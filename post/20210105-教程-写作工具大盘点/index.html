叙言

– <2021-01-05 Tue 13:26>

我的主要工具是CPWS和NWPS, 也就是计算机和笔记本.

我的主要工作是生产和传播知识. 现在的目的是提高知识生产力. 怎么提高? 三方面: 生产者, 生产工具, 生产环境.

  • 内容管理系统 CMS

    1. 内容收集

      1. 观察

      2. 调查

      3. 互联网

    2. 内容生成

      1. 生成者: UGC, PGC, OGC, MGC

        1. UGC, User-generated Content, 用户生成, 比如博客, 维基百科等

        2. PGC, Professionally-generated Content, 专家生成, 比如百科词条

        3. OGC, Occupationally Generated Content, 机构生成, 比如机构公众号

        4. MGC, Machine Generated Content, 机器生成, 比如今日头条的新闻机器人

      2. 生成过程

        1. 素材

        2. 草稿

        3. 排版

        4. 定稿

      3. 生成方式

        1. 著, 创作

        2. 述, 汇总

        3. 编, 编纂

        4. 译, 翻译

        5. 转, 转载

    3. 内容传播

  • 内容管理类型

    1. 网站

      1. 博客

      2. 公众号

      3. 站点

    2. 软件

      1. 印象笔记
